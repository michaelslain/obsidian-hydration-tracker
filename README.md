# Hydration Tracker

A daily hydration tracking plugin for [Obsidian](https://obsidian.md). Track water, coffee, tea, alcohol, and 40+ other drinks with hydration factors, caffeine, and sugar data.

## Installation

Copy `main.js`, `styles.css`, and `manifest.json` to your vault at `.obsidian/plugins/hydration-tracker/`.

## Dev

```bash
bun install
bun run dev            # watch mode
bun run build          # production build
bun run obsidian:debug # open Obsidian with remote debugging on port 9222
bun run start          # watch mode + Obsidian debug together
```
